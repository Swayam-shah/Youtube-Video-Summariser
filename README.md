VidQuery is an AI-powered Chrome Extension that enables real-time interaction with YouTube videos. It uses Retrieval-Augmented Generation (RAG) to analyze video transcripts and answer user queries instantly. Key features include a 'Pop-Out' chat window that stays open for multitasking and smart timestamp links that redirect users to the exact moment in the video where a specific topic is discussed.
